# 🛠️ Tode's PowerShell Toolbox

This is my personal collection of scripts and snippets I use for daily IT tasks. I'm building this out as I work through my Microsoft certs and find new ways to automate things at work.

### What's inside:

* **[Active Directory](./Active-Directory/)** – Quick hits for user management and Entra ID/AD checks.
* **[M365 & Exchange](./M365-Exchange/)** – Mailbox permissions, licensing, and tenant management.
* **[Windows Desktop](./Windows-Desktop/)** – Troubleshooting local machines and fixing common OS issues.

I'm constantly updating these as I learn better ways to handle tickets. If you're here from my LinkedIn or just stumbled across this, feel free to use whatever helps you. 

---
**Note:** Most of these are meant for quick troubleshooting. Just remember to test things before running them on a live production server!
